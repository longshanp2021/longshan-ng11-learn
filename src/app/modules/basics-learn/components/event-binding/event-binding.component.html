属性绑定

Angular中的属性绑定可帮助你设置HTML元素或指令的属性值,使用属性绑定,你可以执行切换按钮,以编程方式设置路径,以及在组件之间共享值之类的功能.

要绑定到元素的属性,请将其括在方括号[]内,该括号会将属性标为目标属性,目标属性就是你要对其进行赋值的DOM属性,
